📁 Scripts Access

The scripts directory is not served by the documentation server for security reasons. To access the script files, please navigate to the project directory: ./scripts/

🚀 Quick Commands

Run these commands from the project root:

  • npm run apim:analyze - Analyze APIM services
  • npm run starwars:characters - Get Star Wars characters
  • npm run docs:generate - Generate documentation
  • npm run sonar:autofix - Run SonarQube auto-fix
